Setup instructions:

Windows

Windows app
  • Install the app
  • Copy {{ name }}.ovpn to C:\Program Files\OpenVPN\config
  • When you launch OpenVPN, it will automatically find the profile and make it available
  • Once OpenVPN is started, initiate a connection by going into the system tray applet and right-clicking on the OpenVPN applet icon
  • Select {{ name }} at the top of the menu and choose Connect
Linux You don't need instructions, haha
OS X

OS X app
  • Install the app
  • Open a Finder window and double-click {{ name }}.ovpn
  • Launch Tunnelblick from the Applications menu
  • Click on the icon in the menu bar at the top right of the screen, then the Connect menu item and choose {{ name }}
Android

Android app
  • Install the app
  • Launch the app and select OVPN Profile at the bottom
  • Tap the plus sign at the bottom and find the {{ name }}.ovpn file
  • Tap Import, then Add
  • Activate the connection by sliding the button next to your profile
iOS

iOS app
  • Install the app
  • Connect your device to a computer
  • Open iTunes on the computer and click on iPhone (iPad etc.) > apps
  • Scroll down to the File Sharing section and click the OpenVPN app
  • Drag the {{ name }}.ovpn file to the OpenVPN Documents window
  • Launch the OpenVPN app on the iPhone. There will be a notification that a new profile is ready to import. Tap the plus sign to import it
  • Activate the connection by sliding the button
